# BizTrac

BizTrac is a mobile-first business management PWA for Ghanaian SMEs. This first pass focuses on UI and UX for authentication, dashboard, inventory, fast sales, expenses, reports, customers, invoices, and business settings using mock data.

## Stack

- React + Vite
- Tailwind CSS
- Lucide React icons
- Recharts for report previews

## Run Locally

```bash
npm install
npm run dev
```

## Current Focus

- Mobile-first app shell with bottom navigation
- Register/login UI and business profile capture
- Dashboard overview with sales, stock, expenses, debtors, and profit
- Inventory cards with boutique fields: size, colour, brand, item type
- Fast new sale flow with cart, payment type, receipt preview, and WhatsApp-ready sharing UI
- Expense form and history
- Reports for daily sales, profit, expenses, best sellers, low stock, and debtors

Supabase schema, authentication wiring, real CRUD, PDF receipts, and offline sync are intentionally left for the next development phase.
